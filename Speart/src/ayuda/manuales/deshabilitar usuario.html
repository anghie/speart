Para Deshabilitar un usuario  se debe haber ingresado previamente al sistema como Administrador Después ir al módulo Usuarios situado en la parte superior izquierda, luego hacer clic en el ítem Lista usuarios.

Seleccionar de la tabla, el usuario a deshabilitar y presionar el botón Deshabilitar, el sistema mostrara  un mensaje indicando lo siguiente



Para agregar las actividades a otro usuario debe presionar en “SI” el sistema le mostrara un msj “Escoja el servidor al que desea pasar las actividades”.

Una vez seleccionado se mostrará si “Desea deshabilitar el usuario”, presionar SI finalmente “Se ha deshabilitado el usuario”
